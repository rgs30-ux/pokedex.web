# 🔴 Pokédex Web

Uma aplicação web interativa que consome a API pública de Pokémon para listar, buscar e favoritar Pokémons em tempo real.

---

## 🚀 Demonstração

👉 Acesse o projeto online:
https://rgs30-ux.github.io/pokedex-web/

## ✨ Funcionalidades

✔️ Listagem de Pokémons
✔️ Busca por nome
✔️ Paginação
✔️ Sistema de favoritos ⭐
✔️ Modal com detalhes do Pokémon
✔️ Interface responsiva 📱

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* API: https://pokeapi.co/

1. Clone o repositório:

```

```

2. Acesse a pasta:

```
cd pokedex-web
```

3. Abra o arquivo:

```
index.html
```
